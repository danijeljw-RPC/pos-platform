# Internationalisation and Industry Templates

## Internationalisation

For APAC, NA, and EMEA expansion.

| Feature | Notes |
|---|---|
| Country-specific config | AU/NZ/SG/HK/UK/US/CA |
| Currency | AUD/NZD/SGD/HKD/GBP/USD/CAD |
| Tax model | GST/VAT/sales tax |
| Date/time formats | Per locale |
| Receipt legal requirements | Per country |
| Language support | Later |
| Tipping rules | US/CA |
| Surcharge rules | AU/NZ |
| Address format | Country-specific |
| Company identifiers | ABN, NZBN, VAT number, etc. |
| Measurement units | Optional |
| Rounding rules | Country/currency |

## Industry templates

Instead of making the user configure everything from scratch, provide setup presets.

| Template | Preconfigured features |
|---|---|
| Cafe | Coffee modifiers, takeaway, GST |
| Bakery | Daily stock, cake orders, GST-free markers |
| Pub/bar | Tabs, split bills, surcharges, tips later |
| Restaurant | Tables, courses, modifiers, KDS |
| Fast food | Quick order, combos, drive-through later |
| Food truck | Offline, event mode, limited menu |
| Retail | Barcode/SKU, returns, inventory |
| Electronics | Serial numbers, warranty |
| Repair shop | Service jobs, quotes, deposits |
| Multi-location chain | Central catalogue, store overrides |
