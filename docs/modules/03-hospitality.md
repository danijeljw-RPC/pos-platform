# Hospitality Features

For cafes, restaurants, pubs, bars, bakeries, and fast food.

## Wishlist

| Feature | Notes |
|---|---|
| Tables/floor plan | Areas: bar, patio, dining room, upstairs |
| Table status | Empty, seated, ordered, served, needs payment |
| Move table | Move order from table 4 to table 7 |
| Merge tables | Group booking |
| Split bill | By item, equal split, custom amount |
| Seat/person ordering | Person 1, Person 2, etc. |
| Course firing | Entree, main, dessert |
| Send to kitchen/bar | Send items to correct prep station |
| Hold/fire items | “Hold mains”, “fire mains now” |
| Bar tabs | Open tab under name/card/table |
| Happy hour pricing | Time/day based pricing |
| Public holiday surcharge | AU/NZ especially |
| Sunday surcharge | Hospitality |
| Service charge | Optional configurable charge |
| Card surcharge | Configurable by payment method |
| Tips/gratuity | Mainly US/CA |
| Kitchen notes | Item/order-level prep notes |
| Wastage recording | Spilt drink, burnt food, staff error |
| 86 item / sold out | Mark unavailable instantly |
| Prep time estimates | Show expected wait time |
| Customer display | Show order/total/payment status |
| QR ordering integration | Optional later |

## Important note

KDS screens are separate devices/sessions. The POS only needs to route items and publish events to KDS/bar/prep stations.
