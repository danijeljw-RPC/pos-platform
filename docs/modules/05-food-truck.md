# Food Truck Features

Food trucks need mobility and offline resilience.

## Wishlist

| Feature | Notes |
|---|---|
| Offline mode | Continue selling during bad internet |
| Local device sync | Orders cached locally |
| Event/location mode | “Saturday market”, “Festival A” |
| Limited menu mode | Smaller menu per event |
| Stock countdown | Only 40 burgers available |
| Quick pay flow | Fast card/cash checkout |
| QR pickup number | Customer order number |
| Portable printer support | Bluetooth/network later |
| End-of-day summary | Per event/location |
| Location-based tax | Important in US |
| Mobile hotspot tolerance | App must reconnect cleanly |

## Event mode

```text
Food truck selects event/location
↓
Menu and prices can be event-specific
↓
Sales are tagged to that event
↓
End-of-day report summarises that location/event
```
