# Kitchen, Bar, Prep Routing and Inventory

Even if KDS is separate, the POS needs routing.

## Kitchen/bar/prep routing wishlist

| Feature | Notes |
|---|---|
| Route by item | Coffee to coffee station |
| Route by category | Drinks to bar |
| Route by modifier | Grilled vs fried |
| Multiple station routing | Burger to kitchen, drink to bar |
| Print docket | If no KDS |
| KDS event publishing | Push to kitchen screens |
| Send/hold/fire | Hospitality |
| Prep status | Pending/preparing/ready |
| Recall ticket | Reprint/resend |
| Void notification | Kitchen sees cancelled item |
