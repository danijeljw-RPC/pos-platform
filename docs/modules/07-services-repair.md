# Services and Repair Business Features

Example: computer repair store, electronics repair, phone repair, service counter.

## Wishlist

| Feature | Notes |
|---|---|
| Service jobs | Create job ticket |
| Customer device record | Laptop, phone, PC, serial number |
| Fault description | “Won’t boot”, “screen cracked” |
| Intake checklist | Charger included? Bag? Password? |
| Quote approval | Customer approves repair |
| Parts used | Add parts to job |
| Labour lines | Hourly/fixed labour |
| Job status | Received, diagnosing, waiting parts, ready |
| Customer notifications | SMS/email later |
| Deposit/payment balance | Pay deposit now, balance later |
| Warranty on repair | 30/90 days etc. |
| Technician assignment | Who is working on it |
| Photos/attachments | Before/after photos later |

## Service job lifecycle

```text
Received
↓
Diagnosing
↓
Quoted
↓
Approved
↓
Waiting for parts
↓
In repair
↓
Ready for collection
↓
Collected / Closed
```
